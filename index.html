Crystal Collector!


Instructions:

  • You will be given a random number at the start of the game (the Target Score).

  • Each of the four crystals has a random value attatched to it. Clicking on a crystal will add that crystal's value to your Score.

  • You win the game if you make your Score equal Target Score. If you go over Target Score, you lose the game.

  • Every time you win or lose the game, Target Score is randomized and the values of each crystal are also randomized.
